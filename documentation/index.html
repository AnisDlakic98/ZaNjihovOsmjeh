Postoji jedan životni fakultet koji studiraju oni koji imaju volju da izmame što više iskrenih osmijeha djeci koja žive bez roditelja. On proizvodi najbolje kadrove iz ove oblasti u našoj zemlji već duže od decenije i ne planira da se zaustavi. Upisuje sve kandidate dobre volje, pa nemojte brinuti da li za vas ima mjesta. Ukoliko želite, samo zakoračite unutra. Lako ćete ga pronaći, na njemu je velikim slovima ispisan grafit NJIHOV OSMIJEH VRIJEDI VIŠE! Ispitni rok je, za omladinu iz svih gradova Crne Gore, svake godine u aprilu!

HTML Struktura

Ova aplikacija koristi Bootstrap  koji je kompatibilan sa svim web pretraživačima (browser-ima) kao što su Google Chrome, Safari, Firefox, Internet Explorer i Opera. Bootstrap je od verzije 2.0 podržavao responsive dizajn. To je značilo da će se sajtovi u Bootstrap-u ubuduće biti dostupni za sve veličine ekrana i da je dizajn web stranica potpuno dinamičan i prilagodljiv. Od verzije 3.0 Bootstrap je usvojio mobile first dizajnersku filozofiju, gde je responsive (prilagodljiv) dizajn postao standardan.

Boostrap 4.3.x Layout

<div class="container">
    <div class="row">
        <div class="col-md-12"> ... </div>
    </div>
</div>

Za više informacija posjetite: http://getbootstrap.com/

Vendor

Direktorijum pod nazivom "Vendor" sadrži sve potrebne JavaScript dodatke za ovu web aplikaciju.

1. Bootstrap (framework za izgradnju prilagodivih, mobilnih web lokacija,)

2. Jquery (JavaScript Biblioteka)

3. Nice Select (jQuery plugin koji zamjenjuje podrazumijevani select)

4. Owl Carousel (jQuery plugin koji omogućuje da napravite prekrasan slajder)

6. Jquery Ui (Kolekcija GUI widgeta, animiranih vizualnih efekata i tema implementiranih s jQuery-om)

Uređivanje i CSS

CSS, JS, HTML datoteke su stvorene na principu modularnosti. Ako vam nisu potrebni svi elementi za vašu web aplikaciju, lako možete ukloniti nepotrebne komponente. Kod je lijepo formatiran i adektavtno odvojen komentarima. To olakšava uređivanje i prilagođavanje dizajna aplikacije. Također, za bolje performanse možete kompresovati (umanjiti) svoje datoteke pomoću internetskih usluga za minifikovanje ili bilo koje druge metode koja vam se sviđa.

CSS minifier - http://cssminifier.com/
JS minifier - https://javascript-minifier.com/
Images minifier - https://tinypng.com/
HTML minifier - http://www.willpeavy.com/minifier/

Izmjena boje

Ukoliko želite da promijenite boju, potrebno je da pristupite fajlu za varijable (_variables.scss) i dodate svoje boje

Primjer:

Izmjena primarne boje - Pronaći red sa nazivom $turquoise-blue: #06aaca;  i dodati željenu boju, npr. $turquoise-blue: #000;

Izmjena Slika

Ukoliko želite da promijenite odredjenu sliku, možete pristupiti folder"img" i jednostvano dodati željenu sliku.

Izmjena fonta

U ovoj aplikaciji su predefinisani sledeći fontovi:
''Poppins', sans-serif' (Avaiable on Google Fonts)

1. "Muli", sans-serif; (dostupan na Google Fonts)
2. "Bebas Neue. cursive", (dostupan na Google Fonts)
3. "Playfair Display, serif", (dostupan na Google Fonts)

Za izmjenu fonta posjetite web stranicu http://www.google.com/webfonts i odaberite željeni font.

Na kraju link morate dodati u HTML ili CSS fajl.

Primjer:

Dodavanje fonta u css fajl:

@import url('https://fonts.googleapis.com/css?family=Poppins:100,200,300,400,500,600,700,800,900');

Dodavanje vrijednosti na body element u css fajlu:

body
{
        font-family: 'Poppins', sans-serif;
}

 

Google Mape

Google ažuriranje 22.06.2019. Zahtijeva da se od sada koriste ključevi API-ja. Za više informacija pročitajte ovaj post - https://googlegeodevelopers.blogspot.com.by/2016/06/building-for-scale-updates-to-google.html To je Google zahtev.

Jednostavno morate unijeti API ključ za svoj widget za Google Maps web mjesto i on će raditi 100% po želji. Da biste kreirali API ključ, idite na ovu stranicu - https://developers.google.com/maps/documentation/javascript/get-api-key i pritisnite plavo dugme „Dobijanje ključa“ i slijedite upustva.

Dodavanje novog API ključa

Pronaći HTML fajl sa sledećom skriptom:

<script src="https://maps.googleapis.com/maps/api/js?key=AIzaSyAsXawwS7Pa4IuJtGnlP kfHzS2cMr4e5OI&callback=initMap"></script>
